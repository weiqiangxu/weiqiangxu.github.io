什么是DHCP

1.Linux的DHCP相关配置

2.如何分配静态IP

DHCP（Dynamic Host Configuration Protocol）是一种网络协议，用于自动分配IP地址和其他网络配置参数给计算机设备。

静态IP地址是由管理员手动分配给设备的固定IP地址，不通过DHCP协议进行动态分配。
